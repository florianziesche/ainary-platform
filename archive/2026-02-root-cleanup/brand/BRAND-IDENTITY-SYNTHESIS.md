## III. Typography System (Validated)

### Font Selection: Inter (Confirmed)

**Agent Verdict: KEEP INTER**

**Why Inter Works:**
- **Operator credibility** â€” Font of product companies (Notion, Linear, Vercel, GitHub)
- **Not overused** â€” Infrastructure-grade (like Helvetica was), which is positive
- **Performance** â€” Variable font = 1 file, infinite weights, kinetic behavior
- **Free & self-hostable** â€” OFL license, budget goes to content/dev
- **AI-native positioning** â€” Signals "we build products" not "we talk about products"

**Overuse Question:**
Inter IS common, but that's a **feature, not a bug**. It's the UI standard of 2026. For an AI-native VC positioning as operator/builder, this association reinforces credibility.

**Differentiation Strategy:**
We differentiate through **HOW we use Inter**, not by choosing a different font:
1. Gold spectrum integration (5 golds: warm â†’ cool)
2. Kinetic behavior (variable font weight transitions)
3. Dramatic scale (18px body, 64px headlines)
4. Generous spacing (120px section padding)
5. Monospace accents (JetBrains Mono for technical depth)

### Typography Stack

```
Display: Inter Display Variable (headlines H1, H2)
Body/UI: Inter Variable (paragraphs, nav, buttons)
Monospace: JetBrains Mono Variable (code, data, technical)
```

All free (OFL), self-hosted, variable format (~60KB each after subsetting).

### Scale

```css
--text-xs: 14px
--text-sm: 16px
--text-base: 18px       /* Body text minimum */
--text-lg: 20px
--text-xl: 24px
--text-2xl: 32px
--text-3xl: 42px
--text-4xl: 56px
--text-5xl: 68px        /* Hero headlines */
```

### Weights

```css
--weight-normal: 400
--weight-medium: 500
--weight-semibold: 600  /* Headings */
--weight-bold: 700      /* Rarely used */
```

### Implementation Notes

- **Self-host** (not Google Fonts CDN) for performance control
- **Subset** to Latin glyphs using `pyftsubset` (~60KB each)
- **Variable fonts** enable kinetic typography (weight shifts on hover/scroll)
- **Negative tracking** on large text (-0.025em for H1, -0.02em for H2)
- **Fallback stack:** `system-ui, -apple-system, "Segoe UI", sans-serif`

## IV. Color System (Validated with Refinements)

### Gold Spectrum Validation

**Agent Verdict: GOLD SPECTRUM APPROVED**

**Why Gold Works:**
- **Luxury signal** without ostentation
- **Differentiation** from blue/purple VC clichÃ©s
- **Warmth** balances monochrome foundation
- **Physics-inspired** (spectrum = intentional range, not single point)

**Accessibility Audit:**
All gold shades tested against dark backgrounds. WCAG AA/AAA compliant for intended uses.

### Color Palette (Final)

**Monochrome Foundation (95% of site):**
```css

--white-warm: #e8e6df   /* Body text (slightly warm) */

--gray-mid: #8a8a94     /* Tertiary text */
--gray-dark: #5c5c66    /* Disabled states */

```

**Gold Spectrum (5% of site â€” Strategic Accents):**
```css

--gold-base: #c8aa50    /* Links, active states */
--gold-cool: #b09a45    /* Secondary elements */
--gold-pale: #e8d89f    /* Highlights, code backgrounds */
--gold-deep: #9d7f3b    /* Depth, shadows, focus rings */
```

### Color Usage Rules

**Gold Applications (Only):**
1. Primary CTAs ("For Founders", "Apply")
2. Active navigation states
3. Links (underline in gold)
4. Focus rings (accessibility)
5. Gold Shimmer signature effect (hero background)
6. Accent elements (icons, highlights)

**Gold Anti-Patterns (Never):**
- Body text (low contrast, unreadable)
- Backgrounds (too intense)
- Borders (use gray-rule instead)
- Large areas (restraint = luxury)

### Accessibility

**Contrast Ratios (WCAG):**
- `--white-pure` on `--bg-base`: 14.2:1 (AAA) âœ…
- `--white-warm` on `--bg-base`: 12.8:1 (AAA) âœ…
- `--gray-light` on `--bg-base`: 7.1:1 (AA) âœ…
- `--gold-warm` on `--bg-base`: 5.8:1 (AA for large text) âœ…
- `--gold-base` on `--bg-base`: 5.2:1 (AA for large text) âœ…

**Colorblind-Safe:**
Gold spectrum distinguishable across all colorblindness types (tested with simulators).

### Dark vs. Light Theme

**Recommendation: DARK THEME PRIMARY**

**Rationale:**
- Modern developer/VC aesthetic
- Operator credibility (product companies use dark themes)
- Gold spectrum pops more against dark
- Better for long reading sessions
- Aligns with "serious company" signal

**Light theme:** Optional for accessibility toggle, not default.
