### Module 1: Foundations (Week 1)
**"Von ChatGPT-User zum AI-Operator"**

- Case Study: 10x Produktivität durch Perspektivwechsel

1.2 Die AI-Operator Mindset
- AI als Mitarbeiter, nicht als Tool
- Governance: Du entscheidest, AI arbeitet
- Compound Knowledge: Jede Interaktion macht das System besser

1.3 Dein erster AI-Mitarbeiter

- Template: "Meet Your AI"

### Module 2: Memory & Context (Week 2)
**"AI die sich erinnert"**

2.1 Das Memory-Problem
- Warum AI alles vergisst
- Die Lösung: External Memory
- File-basierte Persistenz erklärt

2.2 Dein Second Brain aufbauen
- MEMORY.md: Was dein AI wissen muss
- Daily Logs: Kontext der sich aufbaut
- Die goldene Regel: Alles aufschreiben

2.3 Context Engineering
- Wie viel Kontext ist genug?
- Progressive Disclosure
- Template: Memory System Starter Kit

**Deliverable:** Funktionierendes Memory-System

### Module 3: Tools & Automation (Week 3)
**"AI die handelt"**

3.1 Von Chat zu Action
- Tool-Calling verstehen
- Welche Tools dein AI braucht
- Sicherheit: Was darf AI, was nicht?

3.2 Die wichtigsten Integrationen
- Calendar: AI kennt deinen Tag
- Email: Inbox-Scanning
- Web: Research on Autopilot
- Files: Lesen und Schreiben

3.3 Automation ohne Code
- n8n / Make / Zapier Basics
- AI + Automation = Superkräfte
- Template: Automation Starter Pack

**Deliverable:** 3 funktionierende Automationen

### Module 4: Workflows & Systems (Week 4)
**"Deine Personal Operating System"**

4.1 Workflow Design Principles
- Input → Process → Output
- Wiederholbare Muster
- Debugging AI-Systeme

4.2 High-Value Workflows
- Morning Brief (automatisch)
- Research Assistant
- Content Pipeline
- Meeting Prep
- Weekly Review

4.3 Dein individuelles System
- Baue deinen Stack
