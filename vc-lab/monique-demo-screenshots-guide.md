# Screenshot Guide für Monique Demo Backup

## Screenshots die du JETZT machen solltest

### 1. OpenClaw WebChat — Aktive Konversation
**Was:** WebChat mit einer Konversation die zeigt wie du arbeitest
**Wie:** 
1. Öffne http://127.0.0.1:18789
2. Stell eine Research-Frage (z.B. "Give me a summary of today's priorities")
3. Warte auf Antwort
4. **Cmd+Shift+4** → Screenshot speichern

**Speichern als:** `~/Desktop/monique-backup/01-webchat-conversation.png`

---

### 2. Obsidian — Knowledge System
**Was:** Die VC-Intelligence Folder Struktur mit Verlinkungen
**Wie:**
1. Öffne Obsidian (System_OS Vault)
2. Navigiere zu `20-Knowledge/VC-Intelligence/`
3. Öffne eine Datei mit Wikilinks (z.B. Weekly Digest)
4. Zeig den Graph View (Cmd+G) kurz
5. **Cmd+Shift+4** → Screenshot

**Speichern als:** `~/Desktop/monique-backup/02-obsidian-knowledge.png`

---

### 3. Obsidian — Weekly Digest
**Was:** Ein konkretes Digest das zeigt was das System produziert
**Wie:**
1. Öffne `20-Knowledge/VC-Intelligence/Weekly-Digests/2026-W05-Digest.md`
2. **Cmd+Shift+4** → Screenshot

**Speichern als:** `~/Desktop/monique-backup/03-weekly-digest.png`

---

### 4. Terminal — Blogwatcher running (optional)
**Was:** Zeigt dass Feeds automatisch laufen
**Wie:**
1. Öffne Terminal
2. `blogwatcher status` oder zeig die feeds.json
3. **Cmd+Shift+4** → Screenshot

**Speichern als:** `~/Desktop/monique-backup/04-blogwatcher.png`

---

### 5. MEMORY.md — Das "Gehirn"
**Was:** Zeigt wie Kontext persistiert
**Wie:**
1. Im Workspace: öffne MEMORY.md
2. Zeig die Struktur (nicht alles lesen lassen — privacy)
3. **Cmd+Shift+4** → Screenshot

**Speichern als:** `~/Desktop/monique-backup/05-memory-system.png`
