## Website Rules
- White text highlights: max 4-5 key phrases per section
- Filter pills: active = white bg + black text, inactive = transparent + gray border
- Language switcher: "DE" / "EN" (nie "Deutsch" / "English")
- No "coming soon" — add features silently
- Every page with `shared/nav.js` MUST include `.mobile-menu { display:none }` CSS
- Footer: consistent across all pages
- "Ainary" → "ich/I" in prose (Florian IS Ainary). Legal pages keep "Ainary Ventures LLC"
