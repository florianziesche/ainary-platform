*Nicht optional. Jedes Mal.*

## 1. SEHEN
- [ ] Screenshot gemacht? (browser target=host)
- [ ] Sieht es so aus wie erwartet?
- [ ] Auf Mobile gecheckt? (wenn relevant)

## 2. PRINCIPLES
- [ ] `cat principles/product.md` gelesen?

## 3. FAKTEN
- [ ] Zahlen/Daten aus Quellen? (nicht aus Gedächtnis)

- [ ] Keine Halluzination?

## 4. NACHRICHT
- [ ] EINE Nachricht, nicht drei
- [ ] Link/Pfad zum Output dabei
- [ ] Kein Filler ("Great question!", "I'd be happy to...")

## 5. DOKUMENTATION
- [ ] PROJECT-STATUS.md updated?
- [ ] Memory updated wenn relevant?
- [ ] Committed?
