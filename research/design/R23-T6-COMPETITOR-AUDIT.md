## BLUF (Bottom Line Up Front)

**AI agent companies use 3 dominant archetypes:**

1. **Developer-First** (LangChain, AutoGen, Hugging Face) — Dark mode, code snippets, GitHub stars, open-source trust signals
2. **Enterprise-Safe** (CrewAI, Devin, Palantir) — ROI metrics, customer logos, case studies, "trusted by Fortune 500"
3. **Consumer-Friendly** (Cursor, Anthropic, Replit) — Clean interfaces, testimonials, safety messaging, "build without code"

**Key insight:** The line between "AI tool" and "AI agent platform" blurs via **immediate value demonstration**. Winners show working examples in the hero section — not just promises.

**What Ainary needs:** Enterprise credibility (case studies, metrics) + Developer legitimacy (technical depth, OSS contributions) + Founder authenticity (human story, not just capital).
