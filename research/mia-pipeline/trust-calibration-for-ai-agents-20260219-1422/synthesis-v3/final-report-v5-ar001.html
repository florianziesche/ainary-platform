4. The Logit-Free Calibration Gap 84%

(Confidence: High — Technical Finding)

Temperature scaling, the gold standard for neural network calibration, requires direct access to model logits — a capability systematically unavailable through commercial APIs. This single constraint creates the entire technical opportunity space.

Temperature scaling achieves superior performance across multiple architectures, typically reducing Expected Calibration Error by 50–80%[S1]. The method applies a learned scalar parameter T to logit vectors before softmax: softmax(z/T). Simple, elegant, and effective — but it requires the raw probability distributions before the final softmax layer.E

In production environments using commercial APIs from OpenAI, Anthropic, or Google, logit access is systematically unavailable[S1]. This means enterprises cannot apply the most effective calibration method to their deployed AI systems. The gap is not temporary — it reflects deliberate product decisions by API providers to restrict internal model access.E

Budget-CoCoA offers an alternative: generating three independent responses to identical prompts and measuring agreement patterns to infer confidence[S3]. At $0.0005–$0.015 per calibration check[S19], the method is economically viable but sacrifices the mathematical guarantees of temperature scaling.E

5. RLHF Is Destroying Model Calibration 84%

(Confidence: High — Technical Finding)

Reinforcement Learning from Human Feedback systematically destroys model calibration by training models to express unwarranted confidence — and some models suffer permanent damage that no post-training intervention can repair.

RLHF creates a feedback loop that amplifies overconfidence with each training iteration. Reward models consistently prefer confident over uncertain responses, creating gradient updates that increase confidence expressions regardless of actual model uncertainty[S7]. The damage is not incidental — it is structural to how RLHF works.E

The damage bifurcates models into two regimes: calibratable models that retain sufficient signal for post-hoc recovery, and non-calibratable models where logit distributions no longer correlate with accuracy[S30]. No post-training intervention can restore calibration for non-calibratable models, requiring fundamentally different wrapper-based solutions.E

Verbalized confidence adds another layer of complexity. When models express uncertainty through language ("I'm 80% confident that…"), these expressions show systematic bias[S3]. Models overstate confidence in familiar domains and understate it in novel contexts. Adversarial attacks can manipulate verbalized confidence without changing underlying predictions — creating security vulnerabilities in high-stakes applications.E

6. Multi-Agent Calibration Remains Unsolved 84%

(Confidence: High — Technical Finding)

When Agent A calls Agent B, and Agent B calls Agent C, uncertainty propagates non-linearly, breaking every existing calibration framework. No current method addresses this — making multi-agent calibration the highest-value IP opportunity in the space.

Conformal Uncertainty (ConU) provides statistical guarantees for single-model predictions but these guarantees explicitly do not compose across agent boundaries[S9]. If Agent A has 90% confidence and calls Agent B with 90% confidence, the composed confidence is not 81% (0.9 × 0.9) due to complex dependency structures between agents.E

The technical challenge involves three compounding factors: (1) agents may share training data, creating correlated errors violating independence assumptions, (2) context passing introduces sequential dependencies standard methods cannot model, and (3) agent specialization creates domain shifts where calibration valid for one agent may not transfer to another.E

Conformal prediction requires 200–500 calibration examples per model[S9]. A three-agent system with 10 possible actions each would require coverage of 1,000 interaction patterns — making traditional calibration computationally infeasible for complex workflows.E

10. Technical Appendix 87%

Calibration Method Specifications

Temperature Scaling: Applies learned scalar T to logit vectors before softmax — softmax(z/T). Reduces ECE by 50–80%[S1]. Requires logit access unavailable in API deployments.E

Budget-CoCoA: Three independent responses to identical prompts; confidence inferred from agreement patterns[S3]. $0.0005–$0.015 per check[S19]. Works with any API but sacrifices mathematical guarantees.E

Conformal Prediction (ConU): Returns prediction sets guaranteed to contain true answer with specified probability. Requires 200–500 calibration examples[S9]. Guarantees do not compose across agent boundaries.E

RLHF Damage Mechanisms

Reward hacking dynamics create divergence between internal model uncertainty (preserved in logits) and expressed confidence (corrupted by RLHF)[S7]. Calibratable models show monotonic logit entropy–accuracy relationships enabling recovery. Non-calibratable models show no such correlation[S30].E

Multi-Agent Composition Challenges

Three interconnected problems: correlated errors from shared training data, sequential dependencies from context passing, and domain shifts from agent specialization. No current method addresses these[S9].E
