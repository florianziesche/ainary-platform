## TACTICAL RULES ‚Äî The Hard Edges

### Do's
- ‚úÖ Personalize EVERY message (no mass copy-paste)
- ‚úÖ Reference something specific about them (role, post, company)
- ‚úÖ Follow up exactly once after 5-7 days
- ‚úÖ Send messages on weekdays (Tue-Thu is optimal)
- ‚úÖ Track everything ‚Äî what gets measured gets managed
- ‚úÖ Give before you ask ‚Äî lead with value
- ‚úÖ Write in German for DACH contacts, English for international
- ‚úÖ Keep messages under 100 words (short = higher response rate)

### Don'ts
- ‚ùå Never send more than 10 outreach messages/day (LinkedIn flags this)
- ‚ùå Never follow up more than once on cold outreach
- ‚ùå Never lead with "I'm building a company and need..." 
- ‚ùå Never send the same template to people who know each other
- ‚ùå Never pitch in the first message ‚Äî always discovery first
- ‚ùå Never connect + message simultaneously (connect first, message after acceptance)
- ‚ùå Never use LinkedIn automation tools (account ban risk)
- ‚ùå Never message on weekends or after 8 PM

### Response Handling Playbook

| Response Type | Action |
|---------------|--------|
| Positive ("Sure, let's chat!") | Book call within 48h. Send calendar link immediately. |
| Warm ("Interesting, tell me more") | Send 2-3 sentence value prop. Ask for 15 min call. |
| Redirect ("Talk to my colleague X") | Thank them. Contact colleague with "[Name] suggested I reach out." |
| Busy ("Not right now") | Reply: "No problem! Mind if I check back in [2 months]?" Add to nurture. |
| No response (after follow-up) | Move to nurture. Re-engage in 3 months with new context. |
| Negative ("Not interested") | Reply: "Totally understand. Wishing you all the best!" Remove from pipeline. |
