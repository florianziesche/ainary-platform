% Executive Summary
\section*{Executive Summary}

\begin{tcolorbox}[execbox={Kernerkenntnisse}]
\begin{itemize}[leftmargin=*]
  \item \textbf{Marktführer:} Spanflug MAKE (€399/Monat oder €3.990/Jahr)
  \item \textbf{Marktlücke:} Keine Lösung bietet NC-Code-Generierung + Kalkulation in einem
  \item \textbf{Opportunity:} KMU-Segment (10-50 MA) ist unterversorgt
  \item \textbf{Empfehlung:} Positionierung als „Spanflug für kleine Betriebe" + NC-Code-Vorteil
\end{itemize}
\end{tcolorbox}

\vspace{0.5cm}

Der Markt für CNC-Kalkulationssoftware in der DACH-Region wird von wenigen spezialisierten Anbietern dominiert. \textbf{CNC Planner Pro} positioniert sich als kostengünstige Alternative mit einzigartigen Differenzierern:

\begin{itemize}
  \item AI-gestützte Bauteilanalyse
  \item NC-Code-Generierung (einzigartig am Markt)
  \item Betriebsspezifische Anpassbarkeit
  \item Transparente Detailkalkulation
\end{itemize}

\section{Chancen \& Risiken}

\subsection{Chancen}

\begin{enumerate}
  \item \textbf{Unterversorgtes KMU-Segment} — Spanflug zu teuer, TICC zu komplex
  \item \textbf{NC-Code-USP} — Niemand kombiniert Kalkulation + NC-Generierung
  \item \textbf{AI-Welle} — Akzeptanz für „AI macht Vorschläge" steigt
  \item \textbf{Heidenhain-Fokus} — Weit verbreitet in DACH, wenig spezialisierte Tools
\end{enumerate}

% Recommendations
\section{Strategische Empfehlungen}

\subsection{Kurzfristig (0-3 Monate)}

\begin{tcolorbox}[warningbox]
\begin{enumerate}
  \item \textbf{Pilotbetrieb gewinnen} — Erste Referenz für Glaubwürdigkeit
  \item \textbf{NC-Code USP betonen} — In allen Materialien hervorheben
  \item \textbf{Pricing unter Spanflug} — Klare Kostenvorteile kommunizieren
\end{enumerate}
\end{tcolorbox}

\subsection{Mittelfristig (3-12 Monate)}

\begin{enumerate}
  \item Feedback-Loop implementieren — Automatisch besser werden
  \item Werkzeug-API erweitern — Mehr Hersteller einbinden
  \item 5-10 Pilotkunden — Case Studies sammeln
\end{enumerate}

\subsection{Langfristig (12+ Monate)}

\begin{enumerate}
  \item Maschinenpark-Integration — Direkte Anbindung an Steuerungen
  \item Partnerschaften — Werkzeughändler, Maschinenverkäufer
  \item Internationalisierung — AT, CH, dann global
\end{enumerate}

\vspace{1cm}

\begin{tcolorbox}[execbox={Fazit}]
CNC Planner Pro hat eine realistische Marktchance durch:
\begin{enumerate}
  \item \textbf{Klare Differenzierung} (NC-Code + günstiger)
  \item \textbf{Unterversorgtes Segment} (KMU 5-30 MA)
  \item \textbf{Technische Glaubwürdigkeit} (AI-Analyse, Heidenhain-Know-how)
\end{enumerate}

\textbf{Kritischer Erfolgsfaktor:} Genauigkeit der Kalkulation.\\
Ein ungenauer Algorithmus ist schlimmer als kein Algorithmus.
\end{tcolorbox}
