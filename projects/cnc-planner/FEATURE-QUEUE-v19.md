# Feature Queue v19 — Florian Feedback (2026-02-10 22:04)

## Neue Features (aus Florian-Feedback)

### 1. "Frage die KI" Feld (neben Werker + Feedback)
- Freitextfeld wo AV/Werker Fragen stellen können: "Woher kommen die Schnittwerte?"
- Antworten kommen aus der Wissensbasis (REFA, Werkzeugkatalog, etc.)
- Doppelter Nutzen: Hilft dem User UND gibt uns Feedback welche Infos fehlen/unklar sind
- Position: In der Sticky Action Bar neben Werker-Button und Feedback-Button

### 2. Quellen bei KI-Insights
- Jedes Insight im "Pricing Insights" Panel bekommt eine Quellenangabe
- z.B. "Quelle: REFA-Richtwerte 2024, Tabelle 3.2" oder "Quelle: Marktdaten Sachsen Q4/2025"
- Erhöht Vertrauen und Nachvollziehbarkeit

### 3. Berichts-Beschreibungen bei Druck-Buttons
- "Vollständigen Bericht drucken" → darunter: "Enthält: Projektübersicht, Arbeitsgänge, Kalkulation, Zuschläge, Stückpreis, Änderungsprotokoll"
- "Fertigungsanweisung drucken" → darunter: "Enthält: Arbeitsgänge, Werkzeuge, Schnittdaten, Aufspannplan"
- "Kalkulation drucken" → darunter: "Enthält: Zeitkalkulation, Kostenstellen, Zuschläge, Stückpreis"

### 4. Info-Referenz Spalte (aufklappbar)
- Rechts neben jeder Zahl (Bearbeitungszeit, Stundensatz, etc.) ein ℹ️ Icon
- Klick → klappt eine Info-Zeile auf mit:
  - Woher der Wert kommt (REFA, Erfahrungswert, KI-Schätzung, Werker-Input)
  - Referenz (z.B. "REFA Datenblatt Fräsen, Kapitel 4.3")
  - Konfidenz (hoch/mittel/niedrig)
- Nochmal klicken → klappt zu
- Macht die gesamte Kalkulation transparent und nachvollziehbar
