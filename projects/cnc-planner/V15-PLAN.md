## Ziele v15

### 1. Professionelleres UI
- Sidebar-Navigation statt Tabs
- Cleaner Layout (mehr Whitespace)
- Enterprise SaaS Look (wie Notion, Linear)

### 2. Settings als Hauptfeature
- Nicht versteckt in Tab
- Onboarding: "Passen Sie die Werte an Ihren Betrieb an"
- Presets für verschiedene Maschinentypen

### 3. Bessere Informationsarchitektur
- Wichtigstes zuerst (Preis, Zeit)
- Progressive Disclosure für Details
- Klare Call-to-Actions

### 4. Vertrauensbildung
- Datenquellen transparent
- Genauigkeitsangaben
- Validierungs-Badges

### 5. Technische Verbesserungen
- CSS Variables für Theming
- Responsive Design
- localStorage für Einstellungen (bereits vorhanden)
