10. Do / Don't Comparison

Don't

  • Glassmorphism cards (backdrop-filter blur, thick borders)
  • Bold 700 font weight (too heavy, looks cheap)
  • Emoji as icons (ðŸš€ ðŸ’¡ âœ… â€” unprofessional)
  • Too tight spacing (8px between sections)
  • Mix Gold and Indigo as button colors
  • Opacity:0 as default (accessibility issue)
  • Large colored backgrounds (accents are for highlights only)

Do

  • Clean cards (solid bg, subtle border, 16px radius)
  • Semibold 600 font weight (clean, modern, premium)
  • Lucide SVG icons (stroke-width 1.5, consistent)
  • Generous spacing (120px between major sections)
  • Indigo for CTAs, Gold for Badges/Prices (clear separation)
  • Full opacity by default (fade-in animations optional)
  • Accents as highlights (borders, text, small elements)
