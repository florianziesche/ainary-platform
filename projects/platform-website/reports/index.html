Ontologie — Das Ainary Trust System

Epistemisches Layering

Jede Aussage in jedem Report trägt eine Klassifikation. Nicht als Dekoration — sondern damit Sie wissen, wie viel Gewicht eine Aussage verdient.

  • E Evidence — belegt durch externe, zitierbare Quelle
  • I Inference — logisch abgeleitet aus mehreren E-Claims
  • J Judgment — Bewertung basierend auf Evidenz + Erfahrung
  • A Assumption — vorausgesetzt, nicht geprüft

Trust Score

Jeder Report wird automatisch bewertet. Der Score reflektiert Kalibrierung — nicht Überzeugung. Ehrliche Unsicherheit wird belohnt. Überkonfidenz bestraft.

  • 0-30 Untrusted — jede Aussage wird geprüft
  • 31-60 Supervised — Flags werden geprüft
  • 61-80 Spot-Check — 20% Stichprobe
  • 81+ Autonomous — direkter Versand
