Agency Brief: Benchmark Research Summary

ElevenLabs

  • Dark background (#000), bold hero with product demo front-and-center
  • Two-platform split: Creative + Agents -- clear product segmentation
  • Enterprise logos as social proof (Nvidia, Duolingo, Deliveroo)
  • Research timeline showing model evolution builds credibility
  • API-first messaging for developers, separate creative platform for non-devs
  • Startup grants program as alternative to free tier
  • Safety section signals enterprise readiness

Linear

  • Signature restraint: generous whitespace, minimal color
  • Dark theme, subtle gradients, near-monochrome with purple accent
  • Hero shows real product UI, not abstract illustrations
  • 3-pillar value prop: Purpose-built, AI-powered, Designed for speed
  • Interactive product demos embedded in homepage (issues, timelines, chat)
  • Pricing uses 4 tiers: Free, Standard ($8), Plus ($14), Enterprise
  • Feature comparison table with exhaustive detail
  • Changelog section showing velocity and transparency

Vercel

  • Developer-first: code snippets, framework logos, deploy-centric messaging
  • Dark (#000) with high-contrast white text, minimal accent color
  • "Framework-Defined Infrastructure" -- category-creating headline
  • Globe visualization showing global deployment -- impactful visual
  • AI Gateway with live model rankings (builds daily return visits)
  • Pricing: Hobby (free), Pro ($20/seat), Enterprise (custom) -- 3 tiers
  • Extremely detailed usage-based pricing table with per-unit costs
  • Solutions organized by use case, not feature
Key patterns across all three: Dark themes dominate premium SaaS (all three use near-black backgrounds). Product shown working, never as mockups. Social proof via enterprise logos, not testimonials. Generous whitespace signals confidence. CTAs are restrained -- typically 2 per viewport max. Navigation is minimal (4-6 items). Mobile receives full attention with hamburger menus and touch-optimized layouts. All three use sans-serif type systems (Inter, custom fonts). Pricing pages are exhaustive but scannable. Research/changelog sections build credibility and SEO value simultaneously.

For Ainary specifically: The research reports are the differentiator. The website must show actual report output early (like Linear shows real UI). The advisory angle suggests ElevenLabs' enterprise credibility approach. The daily brief creates a daily-visit habit like Vercel's AI Gateway rankings. Pricing should be simple (3-4 tiers) with clear credit-based usage.
