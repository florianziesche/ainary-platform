## I. Gaps Analysis: Current ‚Üí Target

### Typography
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Display font | Cormorant Garamond (serif) | Inter Display (sans-serif) | **REPLACE** ‚Äî serif undermines operator/product credibility |
| Body font | DM Sans (17px) | Inter Variable (18px) | **REPLACE** ‚Äî align with product-company signal |
| Mono font | JetBrains Mono | JetBrains Mono Variable | **KEEP** ‚Äî subset & upgrade to variable |
| H1 size | clamp(2rem,4.8vw,3rem) ‚Üí max 48px | 56‚Äì68px | **INCREASE** dramatically |
| H2 size | clamp(1.4rem,3vw,1.9rem) ‚Üí max 30px | 42px | **INCREASE** |
| Body line-height | 1.75 | 1.7 | Minor tweak |
| Letter-spacing (H1) | none | -0.025em | **ADD** negative tracking |
| Font loading | Google Fonts CDN | Self-hosted, subsetted | **CHANGE** for performance |

### Color
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Background | #0a0f1e (EN), #060608 (DE) | #0a0f1e (standardize) | Unify |
| Gold | Single #c8aa50 | 5-gold spectrum (warm/base/cool/pale/deep) | **EXPAND** |
| Card bg | #12161f | #0f1420 | Minor shift |
| Gold usage | Used on tags, links, mono text, borders | CTAs, links, accents ONLY (5% rule) | **REDUCE & FOCUS** |

### Layout & Spacing
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Max-width | 720px (EN), 640px (DE) | 900px text / 1200px grids | **WIDEN** |
| Section padding | 5rem (80px) | 120px desktop, 80px tablet, 60px mobile | **INCREASE** |
| Hero height | ~auto (padding: 10rem 0 5rem) | 85vh | **EXPAND** to full-viewport hero |
| Content layout | Linear list-based | Grid-based (services, portfolio) | **ADD GRIDS** |
| Section separators | `<hr>` dividers between all sections | Remove most; use spacing | **SIMPLIFY** |

### Navigation
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Style | Fixed top, no glassmorphism | Sticky, glassmorphism, adaptive height | **UPGRADE** |
| Height | ~54px fixed | 72px ‚Üí 56px on scroll | **ADD** adaptive behavior |
| Mobile | `display:none` (nav hidden!) | Hamburger ‚Üí full-screen overlay | **ADD** mobile nav |
| CTA | None | "Get in Touch" gold button | **ADD** |
| Section nav | None | Scroll-based anchor nav below hero | **ADD** |
| Language toggle | Inline nav link | Move to utility area or footer | Relocate |

### Structure
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Hero | Personal intro (email link) | Brand-first (dual CTAs, gold shimmer) | **REDESIGN** |
| About/Vision | Buried at bottom | Promoted to 2nd section | **MOVE UP** |
| Services | Flat offering list | 3-column card grid with gold spectrum | **REDESIGN** |
| Portfolio/Work | Flat offering list | Logo grid (monochrome ‚Üí color hover) | **REDESIGN** |
| Insights | Massive section (clusters, links, quotes, notes) | Preview (3 articles) + CTA to /insights | **CONDENSE** |
| Contact/CTA | "Ask AI" section | Simplified dual CTA | **SIMPLIFY** |
| Footer | Single line "¬© 2026" | 3-column with nav, social, legal | **EXPAND** |
| "Not a Fit?" | Does not exist | Add transparency section | **ADD NEW** |
| Resources | Separate section | Move to /insights or /resources page | **RELOCATE** |
| Perspective | Separate section | Integrate into About/Vision | **MERGE** |
| Newsletter CTA | Inside Insights section | Standalone section or footer | **RELOCATE** |

### Interactions
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Scroll animation | Basic fade-in (16px Y, 0.8s) | Refined (24px Y, 0.5s, threshold 0.15) | **REFINE** |
| Gold Shimmer | Does not exist | Hero background animation (5s loop) | **ADD** signature effect |
| Scroll progress | Does not exist | 2px gold gradient bar at top | **ADD** |
| Card hovers | Basic translateY(-2px) | translateY(-4px) + gold glow | **ENHANCE** |
| Button hovers | None | translateY(-2px) + brightness | **ADD** |
| Nav height transition | None | 72px ‚Üí 56px on scroll | **ADD** |
| `prefers-reduced-motion` | Not implemented | Full support | **ADD** |
| Skip to content | Not implemented | Hidden link for accessibility | **ADD** |

### Accessibility
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Semantic HTML | Minimal (div-based hero) | Full (main, nav aria-labels, section aria-labelledby) | **ADD** |
| ARIA labels | None | On navigation, buttons, hamburger | **ADD** |
| Focus styles | Browser default | Custom gold outline (`:focus-visible`) | **ADD** |
| Skip navigation | None | Skip-to-content link | **ADD** |
| Alt text | Basic | Descriptive, meaningful | **IMPROVE** |
| Landmark roles | None | nav, main, footer landmarks | **ADD** |

### Performance
| Aspect | Current | Target | Change |
|--------|---------|--------|--------|
| Font loading | 3 Google Fonts (CDN, blocking) | 2 self-hosted variable fonts (subsetted) | **OPTIMIZE** |
| CSS | Inline <style> (minified but large) | External stylesheet (cacheable) or critical inline + deferred | **RESTRUCTURE** |
| JS | Minimal (IntersectionObserver) | Modular (IO + scroll progress + nav + shimmer) | **EXPAND** |
| Target weight | Unknown | <500KB initial | **SET BUDGET** |
| Lighthouse target | Unknown | 95+ all categories | **SET TARGET** |
