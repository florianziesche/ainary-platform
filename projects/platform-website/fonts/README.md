# Fonts Directory

## Required Font Files

Download **Inter Variable Font** from the official source and place it here:

**Download Link:** https://rsms.me/inter/

**Required File:**
- `inter-variable.woff2` (Variable font file with weights 300-700)

## Installation

1. Visit https://rsms.me/inter/
2. Download the Inter font family
3. Extract the archive
4. Copy `inter-variable.woff2` to this directory
5. The website will automatically use the self-hosted font

## Why Self-Hosted?

Self-hosting fonts ensures:
- ✅ GDPR/DSGVO compliance (no data sent to Google servers)
- ✅ Better performance (no external DNS lookup)
- ✅ Privacy protection for visitors
- ✅ No dependency on external CDN availability
