# CV Generator — No LaTeX, No Bullshit

## Usage

### Quick (Browser Print)
```bash
open cv-florian.html
# Cmd+P → Save as PDF
```

### Automated
```bash
./generate-pdf.sh cv-florian
# Creates cv-florian.pdf
```

### For Specific Application
```bash
./generate-pdf.sh cv-florian-conviction
# Creates cv-florian-conviction.pdf
```

## Customization

1. Edit `cv-florian.html`
2. Refresh browser or re-run script
3. Done

## Design Principles

- **One Page:** Everything fits on A4
- **ATS-Friendly:** Clean HTML, no fancy layouts
- **Print-Ready:** Looks good printed or as PDF
- **No Dependencies:** Just Chrome

## To Customize for a Fund

1. Copy html file
2. Adjust highlights (top box)
3. Reorder/emphasize relevant experience
4. Generate PDF
5. Send
