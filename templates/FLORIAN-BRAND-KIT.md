# Florian Brand Kit — Visual Standards

*Damit ich beim ersten Mal das richtige Level treffe.*

---

## Visueller Standard

**Benchmark:** McKinsey, top-tier VC decks, Y Combinator
**Nicht:** Canva-Templates, Stock-Grafiken, generische Infografiken

### Was "professionell" für Florian bedeutet:
- Clean, minimalistisch, viel Whitespace
- Keine Emojis, keine Icons (außer sehr dezent)
- Dunkle Farbpalette bevorzugt (#1a365d, #0f172a)
- Typografie: Inter, clean sans-serif
- Keine Gradients außer sehr subtil
- Keine abgerundeten Ecken (oder maximal 4px)
- Zahlen groß und prominent
- Text sparsam — jedes Wort muss arbeiten

### Rote Flaggen (sofort überarbeiten):
- Sieht aus wie "Fiverr-Design"
- Zu viele Farben
- Clip-Art oder Stock-Icons
- Unausgewogene Komposition
- Zu viel Text auf einem Element

---

## Grafik-Workflow (NEU)

### Vor Erstellung:
1. Frage: "Welchen ONE Point soll die Grafik machen?"
2. Referenz suchen: Gibt es eine ähnliche Grafik die Florian mochte?
3. Sprache: IMMER beide (DE + EN) gleichzeitig erstellen

### Format:
- **Output: PNG** (nicht HTML)
- Auflösung: 2x für Retina (2400×1260 für 1200×630)
- Direkt exportieren, keine manuelle Conversion

### Self-Check vor Delivery (muss alle bestehen):
- [ ] Würde das auf einer McKinsey-Slide funktionieren?
- [ ] Ist es auf den ersten Blick verständlich?
- [ ] Keine Emojis, keine generischen Icons?
- [ ] Whitespace ausbalanciert?
- [ ] Text minimal und präzise?
- [ ] DE und EN Version ready?
